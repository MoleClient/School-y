🌐 School-y — Mac Launcher

HTML files open instantly on Mac with no security warnings. Follow the steps below.

1
Open Terminal
Press ⌘ Cmd + Space, type Terminal, press Enter.
2
Paste this command and press Enter
This finds the School-y folder wherever you put it and starts it up.
cd "$(find ~/Downloads ~/Desktop ~/Documents -name 'start.sh' -maxdepth 5 2>/dev/null | head -1 | xargs -I{} dirname {})" && bash start.sh
3
Wait for "Server running" then browse
First launch may take ~30 seconds while packages install. After that it's instant. The browser will open automatically.

Tunnel launcher (for school access)

✲
Start School-y + Cloudflare Tunnel
Gives you a shareable URL for your Chromebook. Open Terminal and paste:
cd "$(find ~/Downloads ~/Desktop ~/Documents -name 'tunnel-start.sh' -maxdepth 5 2>/dev/null | head -1 | xargs -I{} dirname {})" && bash tunnel-start.sh
If the command says "No such file": School-y might be in a different folder. Find the folder in Finder, then drag the start.sh file directly onto the Terminal window and press Enter instead.
