Why tunnel? School filters block known sites. A Cloudflare Tunnel generates a brand-new random URL every session — it's never in any blocklist, and it uses Cloudflare's infrastructure which schools can't fully block.

If you have School-y hosted somewhere that isn't blocked (custom domain, Google Cloud, etc.), enter the URL here. It gets saved so you only need to type it once.

Once saved, opening this file will instantly redirect there every time.

Platforms schools usually can't block:
• Google Cloud Run — uses Google infrastructure
• Vercel / Render / Railway — less commonly blocked than Replit
• Cloudflare Workers — core internet infrastructure
• Custom domain — any platform + an innocent-looking domain name
