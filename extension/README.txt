School-y Chrome Extension
=========================

HOW TO INSTALL (Chromebook, Windows, or Mac):

1. Open Chrome browser
2. Go to: chrome://extensions
3. Turn ON "Developer mode" (toggle in the top-right corner)
4. Click "Load unpacked"
5. Select this "extension" folder
6. Done! Click the School-y icon in your toolbar.

FIRST TIME SETUP:
When you first open School-y, it will ask for a server URL.
Enter the URL of a School-y server (e.g. wss://yourserver.replit.app/wisp/)

If you're running the local server on your own computer, click "Try Local Server".

That's it — no Node.js, no terminal, no commands needed.
